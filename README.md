## How to build for Swift

Execute on terminal:

`make swift`

## How to build for Kotlin

Execute on terminal:

`make kotlin`

## How to build for Java

Execute on terminal:

`make java`

## How to build for Web Assembly

Execute on terminal:

`make wasm`
