## How to build for C++

Execute on terminal:

`make build-cxx`

## How to build for Swift

Execute on terminal:

`make build-swift`

## How to build for Kotlin

Execute on terminal:

`make build-kotlin`

## How to build for Java

Execute on terminal:

`make build-java`

## How to build for Web Assembly

Execute on terminal:

`make build-wasm`
